# Frontend Demo

## Goal

Provide a lightweight dashboard for presenting the project during demos and the final course presentation.

## Planned Views

- project summary and architecture snapshot
- recent backtest metrics
- strategy status and symbol context
- vault share balances and mock NAV
- basic mock interaction panel for deposit/withdraw flows

## Stack Choice

The frontend starts with `Vite + vanilla TypeScript` to keep setup and maintenance light.

## Caveats

The initial scaffold uses mock content and static demo cards. Real artifact loading and wallet/contract integration are future tasks.
